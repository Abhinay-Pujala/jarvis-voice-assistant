# AI-Powered Voice Assistant

A voice-controlled desktop assistant built using Python that can perform tasks such as opening applications, browsing websites, taking notes, performing calculations, and answering questions using OpenAI.

## Features

* Speech Recognition
* Text-to-Speech Responses
* OpenAI Integration
* Open Websites
* Open and Close Applications
* Music Playback
* Voice-Based Calculator
* Note Taking System
* Multi-threaded Command Execution

## Technologies Used

* Python
* SpeechRecognition
* Pyttsx3
* OpenAI API
* Threading

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Example Commands

* Open Chrome
* Open YouTube
* Ask AI what is Java
* Calculate 45 plus 98
* Note complete DSA practice
* Play music
